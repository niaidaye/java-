# super()与 this()的区别

## This():
    1. 作用于本类
    2. 避免发生命名冲突
    3. 通过调用其他重载的构造方法
    4. 优化代码

| 区别点 | this | super |
| :---: | :---: | :---: |
| 属性访问 | 访问本类中的属性，如果没有就从父类中继续查找 | 访问父类中的属性 |
| 方法 | 访问本类中的方法，如果没有就从父类中继续查找 | 直接访问父类中的方法 |
| 调用构造方法 | 调用本类的构造，必须放在构造方法的首行 | 调用父类构造方法，必须放在子类构造方法的首行|
| 特殊 | 表示当前对象 | 无此概念 |
